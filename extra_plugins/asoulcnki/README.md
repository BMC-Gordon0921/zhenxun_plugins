真寻 枝网查重插件

利用 [枝网查重](https://asoulcnki.asia/) 查找最相似的小作文，为防止文字太长刷屏，将内容转换为图片形式发出

### 使用方式

**以下命令需要加[命令前缀](https://v2.nonebot.dev/docs/api/config#Config-command_start) (默认为`/`)，可自行设置为空**

- 查重/枝网查重 + 要查重的小作文

- 回复需要查重的内容，回复“查重”

- 小作文/随机小作文
