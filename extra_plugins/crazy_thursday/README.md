# Crazy Thursday

_🍗 天天疯狂 🍗_

## 版本

v0.2.2

## 安装

1. 丢进插件文件夹

2. 读取文案的路径位于`./resource`下


## 功能

天天疯狂！随机输出KFC疯狂星期四文案。

**重磅新增** 三十余条文案！~~更新假面骑士国配文案~~

## 命令

1. 天天疯狂，疯狂星期[一|二|三|四|五|六|日|天]，输入**疯狂星期八**等不合法时间将提示；

2. 支持狂乱[月|火|水|木|金|土|日]曜日日文触发；
