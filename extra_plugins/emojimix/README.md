# nonebot-plugin-emojimix

适用于真寻 [Nonebot2](https://github.com/nonebot/nonebot2) 的 emoji 合成器

使用方法：

'''
emoji+emoji
示例：😎+😁

'''

若经常下载出错，可以在 `.env.xxx` 文件中设置代理：

```
http_proxy=http://ip:port

```
